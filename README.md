# 尿结石 FTIR 标准品比对与自动初筛系统

第一版 Streamlit 原型，定位为“基于标准品谱图的自动谱型归属与初筛工具”。输出仅供科研/教学参考，不作为临床诊断依据。

## 功能

- 读取合并 CSV，自动识别分隔符。
- 自动判断频率单位：THz 或 cm^-1，并转换到 THz。
- 以 `data-纯品.csv` 的 1–4 THz 频率网格为基准做线性插值。
- 按学长 `最终模型.py` 的参数执行 SNV + SG 二阶导数。
- 支持 mock 规则预测、读取现成结果、预留 joblib 模型接口。
- 输出样品级汇总、重复谱明细、峰位规则、纯品相似度和治疗参考。
- 支持中英文界面。

## 运行

```powershell
cd D:\生医工\机器学习\首次数据及测算\程序设计
python -m pip install -r requirements.txt
streamlit run app.py
```

Windows 也可以在依赖安装完成后双击：

```text
start_app.bat
```

如果当前环境暂时没有 Streamlit，可以先验证核心流程：

```powershell
python run_core_demo.py
```
